
## Lesson Learned: Cron Jobs

**Use `isolated` + `agentTurn` for crons that must execute:**

```json
{
  "sessionTarget": "isolated",
  "payload": {
    "kind": "agentTurn",
    "message": "Run: python3 script.py\n\nIf results, notify David. Otherwise stay silent.",
    "deliver": true
  }
}
```

❌ `systemEvent` to `main` session — may be ignored during heartbeat cycles
✅ `agentTurn` to `isolated` session — spawns separate agent that executes
